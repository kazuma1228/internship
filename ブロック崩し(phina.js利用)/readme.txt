◆phina.jsとは
ゲームが簡単につくれるjsライブラリのこと。
初心者でも使いやすく、少しお作法を勉強すれば動かせるようになります。
ただし、ゲーム制作特有のお作法があるため 慣れ が必要です。

◆jsとは
javascriptのこと。HPやスマホアプリなどの画面描画にアニメーションのような動きを与えるもの。
たくさん利用するとおしゃれな画面をつくれる反面、使いすぎると描画が遅くなることも。

◆ライブラリとは
便利な機能がたくさん用意された図書館のようなもの。
例えば単語の並び替えをする場合、真面目に書くと数百行ソースを書く必要があったりします。
だけど、ライブラリの"並び替えメソッド"を利用すれば1行書くだけで終わったりする!
製造効率が上がるし、有名なライブラリであれば多くの利用者がいるためバグもない(または少ない)。
その反面、利用しようとしているライブラリに既知のバグがないか、利用しようとしているOSやブラウザに対応しているかなど
事前に調べる必要がある。また、デザインを大幅に変更するのが難しい場合が多い。
要するに、利用する場合はそのライブラリが利用シーンにあっているかよく検討する必要がある。
有名どころだと jQuery でよく使います。
